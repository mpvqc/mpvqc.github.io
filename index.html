mpvQC

Home Source Issue Tracker

A libmpv-based application for quick and easy creation of video file quality control reports.

Windows

  1. Go to the release page.
  2. Download the win-x86_64.zip file and extract it.
  3. Double-click on mpvQC.exe to run it.

Linux

  1. Download the Flatpak Ref File.
  2. Double-click on the downloaded file to install it.

For release notes and downloads, see the latest release announcements.
